QUICK SETUP (Choose One)
================================================================================

OPTION A: Claude Desktop Setup
  1. Find your Claude config file (see START-HERE.md for paths)
  2. Copy configuration from claude-desktop-apprunner-config.json
  3. Replace YOUR_TOKEN_HERE with your Fenergo API token
  4. Replace /path/to/ with the full path to apprunner-mcp-connector.js
  5. Restart Claude Desktop
  6. Done!

OPTION B: Local HTTP Server
  Windows:
    .\start-apprunner-connector.ps1 -Token "Bearer YOUR_TOKEN"

  macOS/Linux:
    ./start-apprunner-connector.sh --token "Bearer YOUR_TOKEN"

  Test:
    curl http://localhost:8091/health
